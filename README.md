# 🎵 Music Player in Python  

A simple **Music Player** built using Python that allows users to **play, pause, stop, and select a folder** to play all songs sequentially.  

## 📌 Features  
- 🎼 **Play, Pause, Resume, and Stop music**  
- 📂 **Select a folder** containing `.mp3` files  
- 🔄 **Automatically play all songs in the folder**  
- 🛠 **Built with `pygame` and `tkinter`** for GUI  

## 🛠 Technologies Used  
- **Python**  
- `pygame` (for audio playback)  
- `tkinter` (for GUI)  
- `os` (for file handling)
